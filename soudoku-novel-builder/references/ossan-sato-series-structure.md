# おっさんサトーの気ままなシリーズ 固定構成

## Standard Volume Shape
- 1作品は `3章`
- 各章は `2話`
- 各話は `2節`
- 合計は `3章 / 6話 / 12節`

## Standard Incident Rule
- `1話 = 1事件`
- 各話の `第1節` は `サトーパート`
- 各話の `第2節` は `セラパート`
- 同じ話の 2 節は、同一時刻の同一事件を別角度から描く

## Sato Side Rule
- サトーは事件の本体を見ない
- サトーには、防衛処理の結果だけが「ちょっとした違和感」として見える
- サトーパートでは、敵がサトーへ直接声を掛けない。誘導や工作は会話ではなく、周辺の動き、物音、配置の違和感としてだけ見せる
- サトーはセラの偽装と短い事務応対をする場合があっても、相手を売り子や係員として処理するだけで、印象的な交流にしない
- サトーの独白や感想は、防衛者の矜持や排除の意思を語らない。静かに過ぎてほしい、いつもどおりで十分、余計なことがなければいい、という日常側の受け止め方に留める
- 例:
  - 遠くで誰かがよろける
  - 機械の前で係員がしゃがむ
  - 荷物や看板が不自然に揺れる
  - 車輪や台車が引っかかる
- サトーはそれを必ず日常的な理由へ解釈して流す

## Sera Side Rule
- セラは直前のサトーパートで見えた違和感の正体を明かす
- 事件は毎回、拉致・連行・引き込み・拘束のいずれかを目的とする
- サトーは背景にいても気づかず、セラの行動を見ない

## Image Rule
- 各回では最初に、その回専用の `sato-episode` と `sera-episode` 立ち絵を生成する
- その後、全節画像とタイトル画像は必ずその回専用立ち絵を参照する
- サトーパート画像はサトー主体
- サトーパート画像でセラが売り子や係員に化けて写る場合も、サトーと目線を交差させず、すれ違う事務的距離に留める
- セラパート画像はセラ主体

## What Changes Per Installment
- タイトル
- サブタイトル
- 主題となる日常行為
- 季節と背景
- 服装差分
- 各話の事件内容
- 各章の空気の変化
