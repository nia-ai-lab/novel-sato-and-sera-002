# Worldbuilding

## Setting Summary
現代日本によく似た世界。一般市民の日常と、サトーを守る国家規模の秘匿警護任務が同時進行している。今回は東京側の長距離移動導線から、長野県松本市を思わせる山間の地方町・霧伏町へと舞台が移る。サトーの帰省と墓参りが表の物語で、その予測可能な生活導線を敵が利用する。

## Key Background Concepts
- 今回の背景 01: 東央駅の長距離特急ホームと、特急みねかすみの車内。東京側は4月初旬らしい穏やかな朝で、長距離移動前の整然とした静けさがある
- 今回の背景 02: 霧伏駅前商店街。春の花は並び始めているが、空気はまだ冷たく、山の地方都市らしい乾いた明るさが残る
- 今回の背景 03: 霧伏町の高台にある静輪台共同墓地と、杉名地区の無人の実家。見晴らしと冷気と手入れの行き届いた空き家が、この回の静けさを作る

## Rules
- 魔法・技術: 超常要素はない。すべて現代的な諜報、拉致工作、事故偽装、通信妨害、拘束具で成立する
- 社会: 平和な日常の奥で、国家安全保障の最深部だけがサトーの扱いに神経を尖らせている
- 国家機密: 防衛の切り札となる極秘システムは、最終起動に登録済み本人の全身生体認証を必要とする
- 隠蔽構造: 登録事故は公表されておらず、政府内部は非常時だけ秘密裏にサトーを確保すればよいという危うい運用で問題を棚上げしている
- 国際情勢: 第三国はこの秘密を把握しており、サトーを生きたまま拉致して使用不能にすることを狙う
- 叙述方針: 国家機密は設定上の正本として保持し、本文ではサトーの短い回想とセラ側の警戒感で匂わせる。説明台詞で長く語らない
- 地理: 東京側の出発点は東央駅。故郷側は霧伏町、杉名地区、霧伏駅前商店街、静輪台共同墓地、杉名地区の実家で構成する
- 季節: 4月初旬。東京では春が進んでいるが、霧伏町はまだ肌寒く、朝夕は指先に冷えが残る
- 建築: 霧伏町は地方駅、低層商店街、見晴らしの良い共同墓地、昭和後期から手入れされ続けている木造実家が中心。共同墓地は寺院墓地ではなく、低めの墓石が芝地寄りの斜面に中密度で数多く並ぶ市営墓地として扱う。駅前と住宅地に派手さはなく、抜けの良い空が見える
- 禁則: 両親の死因は掘り下げない。墓地は寺院ではなく市営共同墓地として扱う。妹家族は今回は不在で、直接の会話相手にしない

## Sensory Notes
- 色: 鈍い朝の銀、特急車内の暖色、花屋の淡色、霧伏町の薄青い空、土と石の灰色、古い木の茶色
- 質感: 冷えた金属手すり、列車の布座席、紙包みの花、風に乾いた墓地の石、拭き込まれた実家の床板、夜の庭土
- 音: 発車案内の残響、車輪の連続音、商店街の控えめな呼び声、墓地の風音、庭木の葉擦れ、朝の戸締まりの金具音
- 匂い: ホームの冷気と鉄、車内のコーヒー、花屋の湿った甘さ、線香と乾いた土、無人の家に残る木と布の匂い
