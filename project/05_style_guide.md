# Style Guide

## Narrative Voice
- 地の文: 各話は必ず第1節のサトーパートから始め、第2節のセラパートで同一事件の裏側を返す
- サトーパート: 帰省の移動、無人の実家の整い方、土地ごとの空気差のような「静かな生活感」を丁寧に積む
- サトーパート: 必ず「ちょっとした違和感」を1つ以上入れる。サトーはそれを事件と思わず、日常的な理由へ丸めて流す
- サトーパート: 敵がサトーへ直接話しかけたり、サトーが敵へ応答したりしない。敵の誘導は人の動き、設備、物音、配置の違和感としてだけ見せる
- サトーパート: セラが売り子や係員に化けて視界へ入っても、サトーはその存在を認識しない。許容するのは短い事務応対だけで、目線や間でつながりを作らない
- サトーパート: サトー本人の直接セリフを最低2つ入れる。1つは異変を楽観的に流すセリフ、もう1つはその場の日常の良さを言葉にするセリフで、できる限り役割を分ける
- 会話文: サトーとセラの直接セリフは必ず単独段落にし、前後を空行1行で挟む。Webビルド後も地の文と同段落へ連結させない
- サトーパート: 第1章第1話では、両親の他界した時期と国家機密プロジェクトの記憶を短く回想する。ただし説明口調にせず、発車待ちのあいだの内省として流す
- サトーパート: 東京側の第1章では4月初旬のふつうの朝として描き、寒さの実感は霧伏駅へ着いた第2章第1話で初めて立てる
- セラパート: 直前の違和感の正体を短く鋭く回収する
- セラパート: 敵制圧に刃物を使わない。必要な道具は非刃物に限り、身体操作や環境利用を優先する
- セラパート: 今回は駅業務、売り子、荷捌き、管理作業など「サービス導線への擬態」を鍵にする。第1章のみ売り子偽装を強く使い、第2章以降の基調姿は猫耳メイドへ戻す
- セラパート: セラの台詞は敵への呼びかけや宣告に使わない。防衛が終わったあと、守れたサトーの日常、気分、手触り、余韻だけを短く独白する
- 会話文: 量は多すぎず、空気を支えるために使う。サトーは穏やか、セラは短く辛辣
- テンポ: 日常描写はゆるやかに、護衛処理は短く確実に切る。感傷は引っ張らず、到着後に立ち上がる冷えた空気の描写で保つ

## Reading Experience
- スマホで読みやすい段落長: 2〜4文程度を基本にする
- 1話の気持ちよい流れ: 第1節でサトーの帰省の気分や土地の温度を作り、第2節で読者だけが「守られていた静けさ」の正体を知る
- 次を読みたくなるフック: サトーの控えめな満足と、セラが残した何事もなかった朝の落差

## Image Direction
- タイトル画像: 全話と主要節画像が固まった最後に作る
- キャラクター画像: canonical portraits を元に、その回専用の `sato-episode` / `sera-episode` を先に生成する。サトーは春の帰省服、セラは基本をメイド基調にし、必要な場面だけ `sera-vendor` の売り子差分を使う。`sera-vendor` では上着やコートを足さず、売り子制服そのもので見せる
- 節画像: 第1節はサトー主体、第2節はセラ主体
- サトーパート画像: セラが見えても遠景や遮蔽物の陰に留め、同伴者に見せない
- サトーパート画像: 売り子や係員に化けたセラが写る場合でも、サトーと目を合わせさせず、すれ違うような事務的距離に留める
- セラパート画像: サトーは背景で無自覚、事件の方向を見ない
- 背景画像: 東京側は長距離特急の整然とした人工物、霧伏町側は冷えた空と低層建築、共同墓地の見晴らし、実家の庭の静けさを優先する。共同墓地は低い墓石が芝地状の斜面に中密度で数多く並ぶ市営墓地として描く

## Continuity Rules
- 固定設定: サトーは最後までセラの正体に気づかない
- 固定設定: セラの対処は非致死かつ非刃物に限る
- 固定設定: 1話の2節は同一事件の表裏であり、別事件を混ぜない
- 固定設定: canonical portraits の顔立ちと装備解釈を崩さない
- 固定設定: 墓地は寺院墓苑ではなく、市営共同墓地として扱う
- 後から変えてよい設定: 季節、舞台、題材、服装差分、各話の事件、脇役の顔ぶれ
