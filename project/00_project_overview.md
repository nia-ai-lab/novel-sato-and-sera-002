# Project Overview

## Series Context
- シリーズ名: おっさんサトーの気ままなXX
- 今回の回で固定継承するもの: サトーとセラの基本設定、国家機密の裏設定、サトーは事件に気づかないという構図、3章6話12節の標準構成
- 今回の回で更新するもの: 帰省先の空気、帰省導線、霧伏到着後に立ち上がる4月初旬の肌寒さ、墓参りの感情、列車と地方町での各話事件、今回専用の服装差分
- canonical portraits: project/assets/characters/sato.png / project/assets/characters/sera.png

## Logline
10年ほど前に両親を見送ったサトーが、4月初旬の故郷へ一泊二日の墓参りに向かう。都内の平穏な朝から、霧伏へ降りた瞬間のひやりとした空気までをたどる帰省導線の裏で、かつて彼が率いた国家機密プロジェクトを狙う敵対勢力を、セラが売り子や地元の気配に化けながら静かに刈り取っていく。

## Audience
- 想定読者: 地方帰省の空気、家族の不在が残す静けさ、気づかれない護衛アクションの落差を楽しみたい読者
- 読後感: サトーの帰省がしんみりしすぎず、霧伏で立ち上がる冷えた空気と小さな安堵の余韻が残ること
- 連載ペース: 初回承認で設計とビジュアルを固め、その後は章ごとに確認しながら進める創読形式

## Creative Intent
- この回でやりたいこと: 東京から故郷へ移る空気の変化と、無人の実家で過ごす夜の静けさを中心に、墓参りの私的な時間を壊さない護衛劇を作る
- 絶対に入れたい要素: ホームでの回想、特急車内の売り子偽装、花を買う駅前商店街、見晴らしの良い市営共同墓地、無人の実家の庭、翌朝の冷えた出立
- 避けたい要素: 両親の死因の説明過多、過度な悲壮感、露骨な陰謀説明、刃物や流血に寄る見せ場

## Product Intent
- 創読としてどんな体験にしたいか: サトーの帰省を一緒にたどりながら、各話のセラパートで「守られていた静けさ」の正体がひとつずつ明かされる構成にする
- 画像生成の方針: canonical portraits を元に、この回専用の `sato-episode` と `sera-episode` を先に生成する。セラは「売り子に化ける」回らしく、サービス導線に紛れ込める春の偽装を基調にする
- モバイル Web での見せ方: 車窓、駅前、墓地、高台、庭先の抜け感を横長ビジュアルで見せつつ、縦スクロールでは冷気と余韻がつながる流れを作る
