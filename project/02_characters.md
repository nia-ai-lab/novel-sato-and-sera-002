# Characters

## Cast Rules
- シリーズ共通の正本キャラクター画像は project/assets/characters/sato.png と project/assets/characters/sera.png
- 各回では canonical portraits を元に、その回専用の sato-episode / sera-episode を先に生成する

- 画像生成後は見た目の定義をここに固定する

## Main Cast

### character_id: sato
- 名前: 佐藤一（さとうはじめ）。本文中の表記はサトー
- 役割: 表の主人公。東京在住の中年男性
- 年齢・立場: 45歳。バツイチ独身、子なし。現在は民間の監査・資料整理系の仕事をしている
- 秘匿経歴: かつて国家防衛の極秘システム開発責任者だったが、本人は終わった仕事として扱っている
- 外見: 身長175cm前後。やや長身で細マッチョ。黒髪長髪を後ろで束ね、口ひげと顎ひげがある
- 共通性格: 温厚、几帳面、観察眼はあるが危険への想像力だけが日常寄り
- 共通話し方: 落ち着いていて丁寧。少し自虐的
- 共通弱点: 違和感を平和な理由へ解釈して流しやすい
- 今回の欲望: 両親の墓参りをきちんと済ませ、無人の実家で一晩だけでも肩の力を抜いて過ごすこと
- 今回の恐れ: 両親の家と記憶が「もう用事だけ済ませる場所」になってしまうこと。本人は意識していないが、かつての防衛プロジェクトに気持ちを戻されることも避けたい
- 今回の服装・持ち物: 4月初旬の冷えに合わせた着込んだアメカジ寄りの春装。くたびれたワークジャケット、濃色デニム、革靴、実家へ持ち込む小ぶりの旅行鞄。墓参り用の花は霧伏駅前で買う
- 今回の変化: 両親が他界した時期と、当時自分が率いていた国家機密プロジェクトの記憶が、東央駅のホームで不意につながる。本人は「今はどうなっているのだろう」と一瞬思うだけで深追いしない
- 画像プロンプト要約: canonical portrait の顔立ち・体格・髪型・ひげを厳守しつつ、肌寒い春の帰省に合う落ち着いた旅行服へ更新する

### character_id: sera
- 名前: セラ
- 役割: 裏の主人公。サトーを秘匿護衛する実働担当
- 年齢・立場: 見た目は20代半ば。内閣系秘匿警護部門の現場要員
- 任務前提: サトー本人に真相を知らせず、敵対勢力による拉致と連行を導線ごと阻止する
- 外見: 身長160cm前後。銀灰色の髪、金色の目、しなやかな体つき
- 共通装備: 猫耳カチューシャ型の警護支援ユニットを常時装着する
- 共通性格: 忠実、冷静、毒舌気味。サトーの小さな幸福を本気で守っている
- 共通話し方: 短く要点だけ。独り言でだけ感情が少し漏れる
- 今回の欲望: サトーの帰省を単なる移動任務にせず、墓参りと実家の一夜を静かなまま完了させること
- 今回の恐れ: 予測しやすい帰省ルートのため、敵が「生活導線の自然さ」を利用して接触しやすくなっていること
- 今回の服装・持ち物: 基本はデフォルト寄りのメイド基調警護装束を春の寒さに合わせて調整する。猫耳ユニットを残した整ったメイドシルエット、小型ポーチ、隠し拘束具と通信阻害具を携行する。第1章の売り子偽装はコートを重ねず、車内販売の制服そのものだけで成立させる
- 今回の変化: 第1章だけは列車導線に合わせて売り子偽装を使うが、第2章と第3章ではデフォルトの猫耳メイド姿へ戻る。今回の護衛は「気配を置かないこと」より「その場の導線に自然に存在していること」を優先する
- 画像プロンプト要約: canonical portrait の顔立ち・髪色・猫耳装備を厳守しつつ、基本は春のメイド基調、必要な場面だけ売り子差分を派生で持つ

## Supporting Cast
- 妹夫婦とその子ども二人: 杉名地区近くに住み、現在は無人の実家を定期的に手入れしている。今回は都合が合わず不在のまま、家の整い方だけで存在を感じさせる
- 敵役: 駅係員、車内販売、荷捌き担当、墓地管理、近隣送迎など、各地点の業務導線に紛れ込む偽装で統一する
- 固定の脇役を出す場合も、サトーの日常の気分を壊さない範囲に留め、妹家族は今回は直接登場させない
